# News Wave
## A Fullstack Website
### Technologies Used:
#### Front End 
* React JS
* React Routers
* Tailwind CSS
* Material UI (in register, login, articles card and add pages)
* Daisy UI (in homepage)
* React Icons
* Lottie React (For Animation)
* Firebase (for authentication)
* React Icons
* Axios
* Tanstack Query

#### Backend
* MongoDB
* Express JS
* CORS

### Features of the website
* Awesome sliders
* User Friendly UI
* Awesome sweet alerts
* Admin Panel
* Fully device responsive
* Free Users and Premium Users Section
* Stripe payment method system
* Eye catching UI
* Sticky Navbar
* Password Authentication
